# Student-Grade-Analyzer
This is a simple data analysis project built using **Python and NumPy**.   It simulates student exam grades and performs basic statistical analysis such as averages, passing/failing students, and top performers.
## 🎯 Project Purpose

The goal of this project is to practice:
- NumPy array operations
- Vectorized computations (without loops)
- Boolean masking
- Basic data analysis logic

---

## 📌 How It Works

Each student has:
- Midterm (vize)
- Final exam

Final grade is calculated using:

- 40% Midterm
- 60% Final

---

## 🧮 Formula

Final Score:

```text
score = 0.4 * midterm + 0.6 * final
